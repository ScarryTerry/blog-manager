## Installation

```bash
$ npm install
```

## Running the app

```bash
# run production mode (.production.env file configuration required)
$ npm run start

# run  dev mode (.development.env file configuration required)
$ npm run start:dev


#Please, use the following contants for your .env file
# 
#.env configuration: (.env || .development.env || .production.env)
#PORT= application port
#MONGO_USER= mongobd username
#MONGO_PASSWORD= mongobd password
#PRIVATE_KEY= your secret key for JWT auth
